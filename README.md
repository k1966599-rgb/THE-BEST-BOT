# THE-BEST-BOT

This is a Telegram trading bot that uses Elliott Wave analysis and other indicators to identify trading opportunities.

## Phase 1: Basic Telegram Bot

This initial version sets up the basic Telegram bot and provides a welcome message.

### Setup

1.  **Clone the repository.**
2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

### Running the Bot

1.  **Run the bot script:**
    ```bash
    python bot.py
    ```
2.  **Interact with the bot on Telegram:**
    - Open Telegram and find your bot.
    - Send the `/start` command to receive the welcome message.
